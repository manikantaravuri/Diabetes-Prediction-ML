- Scikit-learn
- Matplotlib
- Google colab

1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training using SVM
5. Model Evaluation
6. Prediction System

The model was trained and evaluated using classification metrics such as accuracy score.
